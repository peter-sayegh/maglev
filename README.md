# Maglev
This repository contains the code and experimental data used in the paper "Dynamic Stability and Levitation Control in Maglev Systems" by P. Sayegh, H. Déo, Y. Abou Rabii, and Pr. A. Couairon.

We present an experimental, project-based approach to teaching classical electrodynamics through the construction and analysis of a scaled-down magnetic levitation train. Using flexible magnetic tape as a guideway, we demonstrate stable levitation and perform precise measurements of the vertical gap as a function of load. These experimental data are validated using an analytical force model derived from a dipole-layer approximation, which confirms the self-restoring vertical stability of the repulsive configuration. By bridging laboratory measurements with theoretical energy methods, students can extrapolate their scaled results to industrial parameters, successfully reproducing the passenger capacity of a full-scale commercial Maglev vehicle. This project offers a tangible platform for applying core concepts such as magnetic field geometry, vector potentials, and Earnshaw’s theorem to a high-interest, real-world engineering challenge.
